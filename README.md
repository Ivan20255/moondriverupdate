# moondriver

Moon Express Inc. - Driver Trip Sheet Modernization.

A high-fidelity, interactive, and mobile-friendly digital version of the Moon Express trucking form.

## Features
- **Smart Data Entry**: Zip-to-city lookup and city-to-zip autocomplete.
- **Auto-Calculations**: Real-time mileage and expense totals.
- **High-Quality Export**: Generate and send trip reports as high-quality JPEGs via email.
- **Custom Branding**: Pristine SVG logo matching Moon Express Inc.
- **Persistence**: Auto-saves form progress locally.

## Development
To run locally or on Replit:
1. Open `index.html` in any browser.
2. Or run `npm start` if you have Node.js installed.
